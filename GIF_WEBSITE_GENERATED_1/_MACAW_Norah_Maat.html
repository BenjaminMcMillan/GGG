MACAW

The Good

The program is very similar to the Adobe programs. Elements can snap to the grid lines in order to keep everything clean. It simplified many aspect of CSS styling. The controls are easy to understand and adjust. Macaw also provides a preview feature.

The Bad

Everytime you open the program it gives you a new blank file. Online documentation is hard to find and some features are confusing. The features the program offers are very basic and so are the font choices that are very similar to each other.

The Bottom Line

Macaw offers are simplified version of CSS styling and HTML making. It can be a great tool for laying a fast basic foundation for a website that you can later on alter in another coding program to add your own style.
