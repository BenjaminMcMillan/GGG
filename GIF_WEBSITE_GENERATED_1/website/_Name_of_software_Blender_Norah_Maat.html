Name of software Blender

The Good

There are a lot of options to have fun with. It's a completely free tool for 3D modeling, rendering and a lot of other cool stuff.

The Bad

It takes a long time to learn as a beginner. Looks complicated.

The Bottom Line

Professional paid 3D programs will give you more options and higher level of work. The interface is quite hard to understand and to get used to. If you do not take the time to learn it you get frustrated easliy since it does not provide explanation on where to begin.
