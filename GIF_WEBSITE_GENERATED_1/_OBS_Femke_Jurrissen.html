OBS

The Good

Very useful tool for designing some interface for your livestream. You are able to add filters, text, images, videos, etc. These will appear as a sort of layer on top of your camera input. You can also connect the software to streaming platforms, so you are able to stream the design directly to your livestream.

The Bad

Very difficult interface. Not very practical. You really have to become familiar with the different functions inside this program to be able to use it efficiently.

The Bottom Line

Free and open source software for video recording and live streaming.

Rating

4,5
