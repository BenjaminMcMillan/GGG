Sip

The Good

It’s nice that you can pick a color from any pixel that you can find on your screen with some sort of magnifier. It saves the color immediately in your history and you can create palettes’, also you can set in what type of color code you want to receive this specific color, css hex f.e. or openGL, javaRGB f.e. there are many. There’s a size and zoom on the magnifier and, the amount of colors you want to pick, many settings to change to make sure you get the right color. What is veeeery nice tho is that when you’ve picked your color, it’s automatically saved on your clipboard, so when you ‘paste’ it’s the code of the color. Which is very handy.

The Bad

The fact that you have to pay 10$, it’s only once so it’s not too expensive, but I think there are more ways to pick a specific color from screen that are free. I don’t feel like personally I need this app THAT bad to find specific colors, I can also use photoshop. And not really interested in sharing them too.

The Bottom Line

It just depends per person how many times you feel like finding out which color has been used. This app is very simple and just does exactly what you want it to do, which is great. Nothing more, nothing less. I don’t like the fact that you have to pay, but hey, they also made the app and need to make some $$ which is understandable.
