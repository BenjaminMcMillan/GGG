Coolors

The Good

This software is made to design a 5 color palette according to your own preference. The process is very simple. By pressing the space bar the colors change and you can block colors and define them by the options Coolors provides. There are infinite combinations of colors and tones. The combinations that you created can be exported and shared in the library. There is also a option for different forms of color blindness. This way you can see colors that only people with this form of blindness can see.

The Bad

The application has very few options to generate combinations, only allows you to create basic color combinations of 5 color ranges.

The Bottom Line

This tool is not only for graphic designer but also for photographers, illustrators, set designers, fashion designers etc.
