Sculptris

The Good

1. There’s a explanation on keys immediately on the right, also every button states in the interface says what it’s for 2. Symmetry mode, if you have a symmetrical sculpture, you just need to create one side. 3. When Sculptris froze and I had to restart the program it was still where I left it, so luckily didn’t need to start from stratch again!

The Bad

1. The interface (probably as it’s free) looks quite pixelated 2. There is a small introduction on keys on the bottom right, but that’s about it. -- this means you kind of end up pressing all   buttons to figure out what it exactly does  (or you can watch a youtube tutorial) - It does state ‘read more in the documentation’ but not where to find this

The Bottom Line

It’s hard!!!!!!! It might be a handy programme when you are already quite skilled in these type of sculpting programmes, but it definitely has a hard learning curve. Some of the keys are too sensitive and it’s hard to work very precise. Trying to find the documentation is hard and it seems that Z-Brush is actually the newer, better version of Sculptris. There is way more advertising for this programme than Sculptris actually has, although it’s weird that they don’t really talk about the differences of Z-Brush and Sculptris, which one is better- the way to do it.
