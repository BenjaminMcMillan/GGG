Wix.com

The Good

It is very easy to use and you don’t need to have coding skills to create your website. You can choose for a standard template, you can choose and asjust a template to your own liking and you can create a template by using one of the many design options. Another handy thing about wix.com is that you can find many tutorials on youtube.

The Bad

When using the free version, you find sentence “Proudly created with Wix.com “ in the footer of your site. It opens many new tabs.

The Bottom Line

Great tool for creating a site without having to spent hours on it. Only downside is the price. You kind of need to pay a lot if you want all the optional options.

Rating

4
