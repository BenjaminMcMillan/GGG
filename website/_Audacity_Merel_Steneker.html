Audacity

The Good

Really easy to use for quick audio sketches without the need to go to the AV studio for equipment. All useable for both Mac and Windows which makes filesharing easier.

The Bad

The interface is not user friendly and feels crowded. Working in the application it stays hard to keep things organised.

The Bottom Line

Useful for small experiments but becomes insufficient when working in bigger projects or files.
