Tinkercad

The Good

The interface is really easy to use allowing the user to do a huge range of posibilities without any effort. The program is quite simple, you just need a few days to get used to tinkercad. It provides STL files which can work with a 3D printing slicer program. The tutorial is really good to get started in tinkercad. The fact that it's entirely online means there's no software to license or install.

The Bad

Sometimes it can be difficult to follow the dimensions. However, once you get used to the program it becomes a lot easier. While the program is fairly intuitive, this is one area that takes a little practice. There are limits to what you can design with this program however. Because this program uses geometric shapes, it can be difficult to get really creative because you can’t really draw freeform shapes to exactly what you want to draw. You have to understand how to do it within the provided forms and limitations of the software. It doesn’t work offline.

The Bottom Line

Tinkercad provides a pleasant way of creating beautiful 3D models and designs right inside your browser which is suitable for any age group. Enjoy many features and a large community in this complete experience.
